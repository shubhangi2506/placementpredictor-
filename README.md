# 🎓 AI Student Placement Predictor

A Machine Learning web application that predicts student placement chances using CGPA and IQ scores.

## Features

* Predict placement outcome
* Placement probability estimation
* Dataset visualization
* Decision boundary visualization
* Interactive Streamlit interface

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Streamlit

## Machine Learning Model

* Logistic Regression
* Accuracy: 85%

## Project Structure

placementpredictor/
├── app.py
├── model.pkl
├── placement-dataset.csv
├── requirements.txt
└── README.md

## Run Locally

pip install -r requirements.txt

streamlit run app.py
